# Product Overview

Project name: EduGate (working name)

EduGate is a mobile educational game that converts children's screen time into learning opportunities.

The application periodically triggers short educational quizzes during device usage. These quizzes take 5–20 seconds to complete and are gamified with rewards such as XP, coins, streaks, and unlockable items.

The system is built with a family account structure where a parent is the primary account holder and can create multiple child profiles for siblings.

Each child profile maintains independent:
- learning history
- progress
- levels
- achievements
- statistics

Primary goals:
- encourage micro-learning habits
- reduce unproductive screen time
- provide parents with measurable learning insights
