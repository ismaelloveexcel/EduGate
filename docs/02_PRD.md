# Product Requirements Document

## Problem

Children spend large amounts of time on mobile devices with little educational value.

Parents struggle to control screen time without constant supervision.

## Solution

Introduce a learning interruption system that triggers micro quizzes during phone usage.

These quizzes reinforce learning while maintaining a game-like experience.

## Target Users

Primary users:
Parents with children aged 6–14

Secondary users:
Children who interact with the quiz game.

## Key Features

Parent
- parent login
- create child accounts
- configure quiz frequency
- view analytics

Child
- child profile login via PIN
- quiz gameplay
- rewards and progression
- avatar customization

System
- adaptive difficulty
- progress tracking
- push notifications
- analytics

## Success Metrics

Daily active users  
Quiz completion rate  
7-day retention  
Average quizzes per child per day
