# MVP Scope

## Included

Parent features
- Parent account login
- Create child profiles
- Manage quiz settings
- Parent dashboard

Child features
- Child PIN profile
- Quiz gameplay
- XP and streaks
- Coins rewards
- Level progression

System features
- Firestore data storage
- Quiz attempt tracking
- Push notification quiz triggers

## Not Included (Phase 2)

- Multiplayer
- AI generated quizzes
- School integrations
- Homework scanning
- Social features
- Ads
