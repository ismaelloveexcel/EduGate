# Product Requirements Document

## Product Name
EduGate (working name)

## Product Vision
Transform children's screen time into learning time through short educational quizzes delivered during phone usage.

## Core Concept
Children answer micro-quizzes every 15–30 minutes while using their phone. The experience is gamified with XP, coins, streaks, and levels.

## Account Model
Parent = main account  
Parent can create multiple child profiles (siblings).

Each child profile has:
- PIN login
- independent progress
- XP, streak, coins
- quiz history

## Core Features
- Parent login
- Child sub-accounts
- Quiz engine
- Notification triggers
- Gamification
- Parent dashboard analytics

## Target Audience
Parents with children aged 6–14.

## Platform
Flutter mobile app (iOS + Android)

Backend:
Firebase (Auth, Firestore, FCM, Analytics)
