Target feeling:
Fun + playful + educational.

Colors:
Primary: #4F46E5
Secondary: #22C55E
Accent: #F59E0B

Design style:
- bright
- friendly

UX rules:
- quizzes must take less than 20 seconds
- minimal reading required
- large buttons
- animations for rewards
